# SneakeRevolution
As a sneaker collector & re-seller keeping track of my collection can be tricky from the day to day.
I created a platform via which sneaker enthusiasts can display/store data regarding their collection.

### Technologies Used
- HTML
- CSS
- JavaScript
- jQuery
- MongoDB
- Mongoose
- Express
- EJS
- Express Sessions
- Bcrypt
- Morgan
- Method-Override
- Trello
- Mockflow
